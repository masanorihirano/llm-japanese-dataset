# 日本語 WordNet
## 作成方法
https://bond-lab.github.io/wnja/ から取得
```
wget "https://github.com/bond-lab/wnja/releases/download/v1.1/wnjpn.db.gz"
gunzip wnjpn.db
poetry run python generate.py
```
